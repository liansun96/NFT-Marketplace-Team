
# NFT Marketplace

This project is only frontend UI design using React.


## System Requirements

- Node 18.0+
## Tech Stack

**Client:** React, Framer motion,TailwindCSS

## Run Locally

Clone the project

```bash
  git clone https://github.com/liansun96/NFT-Marketplace-Team.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
